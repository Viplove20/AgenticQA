## 4.3 Cart Item Actions

### 4.3.1 Remove Item
```
Click Trash Icon (×) →
  Show Confirmation (optional):
    "Are you sure?" [Cancel] [Remove]
  →
  DELETE /api/ecom/cart/remove-from-cart/{productId}
  →
  Remove from UI with fade animation
  →
  Recalculate totals
  →
  Show toast: "Item removed from cart"
  →
  Update cart badge
```

### 4.3.2 Buy Now (Individual Item)
```
Click "Buy Now" on item →
  Validate cart
  →
  POST /checkout/initiate
  →
  Redirect to /checkout?productId={id}
  →
  Show checkout form with item details
```

**Test Locators:**
```javascript

```

## 4.4 Cart Summary & Checkout

### 4.4.1 Cart Totals Calculation

```javascript
Subtotal = Sum of all (item.price × item.quantity)
Tax = Subtotal × 0.10 (if applicable)
Shipping = 0 (if subtotal >= 5000) else "Contact"
Total = Subtotal + Tax + Shipping
```

**Test Locators:**
```javascript
const subtotal = 'span.value:contains("Subtotal")';
const total = '.totalRow.final .value';
const checkoutButton = 'button:contains("Checkout")';
```

### 4.4.2 Checkout Button
```
Click "Checkout" →
  Validate cart has items
  →
  POST /checkout/initiate
  →
  Redirect to /checkout
  →
  Display payment form

Disabled if: Cart empty
Loading: Show spinner during processing
```

## 4.5 Continue Shopping Button
```
Click "Continue Shopping" →
```
